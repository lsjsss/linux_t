# Single Choice
1、对shell脚本中使用函数描述错误的是_____。
A.将重复指令定义成公共语句块 
B.使脚本代码更加简洁，增强易读性 
C.函数的定义语句必须出现在调用之后 
D.提高Shell脚本的执行效率 
【答案】**C**
